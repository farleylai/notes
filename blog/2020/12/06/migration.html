Seamless Deployment on GitHub Pages

Built with fastpages, an easy to use blogging platform with extra features for Jupyter Notebooks.

It automates the process of creating blog posts via GitHub Actions, so there is no need to fuss with conversion scripts. In summary, fastpages facilitates automatic deployment with GitHub Actions. A dedicated <username>.github.io repo is no longer necessary.

Editing on Google/Colab with Access to GPU/TPU

Each post can be either in Jupyter notebook or markdown. If in Jupyter notebook, the editing and running are well integrated with Google/Colab, implying interactive computation with access to GPU/TPU for free as follows.
